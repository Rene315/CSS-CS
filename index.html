CSS Box Model Reference

Content Area Properties

Property Name Description
box-sizing Box Sizing Controls whether the padding affects the specified width and height.
width Width Sets an elements width.
min-width Minimum Width Sets the minimum width of an element.
max-width Maximum Width Sets the maximum width of an element.
height Height Sets the height of an element.
min-heigth Minimum Height Sets the minimum height of an element.
max-height Maximum Height Sets the maximum height of an element.

Padding Area Properties

Property Name Description
padding-top Top Padding Sets the padding height on the top of an element.
padding-right Right Padding Sets the padding width on the right of an element.
padding-bottom Bottom Padding Sets the padding height on the bottom of an element.
padding-left Left Padding Sets the padding width on the left of an element.
padding Padding Shorthand property that sets the padding on all four sides of an element at once.

Border Area Properties

Property Name Description
border Border Shorthand property that sets an element's border-width, border-style, and border-color
border-width Border Width Shorthand property that sets the width of an element's border.
border-style Border Style Shorthand property that sets the line style on all four sides of an element's border.
border-color Border color Shorthand property that sets the color of an element's border.
width Width If box-sizing is set to border-box then this property sets the width of the border area.
height Height If box-sizing is set to border-box then this property sets the height of the border area.

Margin Area Properties

Property Name Description
margin-top Top Margin Property that sets the margin area on the top of an element. Positive values place the element farther from neighbors and negative values bring it closer to its neighbors.
margin-right Right Margin Property that sets the margin area on the right of an element.
margin-bottom Bottom Margin Property that sets the margin area on the bottom of an element.
margin-left Left Margin Property that sets the margin area on the left of an element.
margin Margin Shorthand property that sets the margin area on all four sides of an element at once.
